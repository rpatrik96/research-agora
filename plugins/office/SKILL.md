# Office Skills

Microsoft Office document creation skills (PowerPoint, Word, Excel).

## Available Commands

| Command | Description |
|---------|-------------|
| `/pptx-create` | Create and edit PowerPoint presentations |
| `/docx-create` | Create and edit Word documents |
| `/xlsx-create` | Create and edit Excel spreadsheets |

## Installation

```bash
/plugin install office@research-agora
```
